Linux下安装Nginx

发表于 2017-09-08 | 分类于 linux

准备工作

系统条件

系统本身要已安装gcc-c++才能进行源码编译安装，如没有，基于生产环境的要求，可参照如下方式进行安装

挂载光盘

配置本地yum仓库源(/etc/yum.repos.d/local.repo)

刷新本地仓库源

安装gcc-c++

准备安装包

  • nginx：nginx源码包
  • zlib：用于支持传输时压缩解压缩
  • openssl：用于支持加解密
  • pcre：用于支持正则表达写法

安装工作

预配置

将几个源码包放到同一个目录，并进行解压缩，进入解压缩后的nginx源码目录进行预配置（–prefix指定安装目录）
